Observables

An observable is a stream of data that can emit multiple values over time.

Cold Observable

Starts on subscribe, example: Observable, of()/p>

Hot Observable

Already producing values, example: Subject, EventEmitter

Observer

An observers is an object with methods next, error, and complete that receives notifications from observable.

Subscription

You subscribe to an observable to receive values.

Subject

Subject is a special type of observable that allows multicasting, which means it can act as both an observable and an observer. There are 4 types of subjects.

Subjet

A basic type, which emits values to all active subscribers.

Behaviour Subject

It requires initial value, and emits latest values to new subscribers.

Replay Subject

It caches and replays a specified no. of previous values to new subscribers.

Async Subject

It emits last value to all subscribers after completion.

Operators

Operators are functions that perform operations on stream of data using pipe.
